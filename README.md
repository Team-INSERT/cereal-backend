# Cereal - 교내 랜덤 채팅 어플

## See - Real!

씨리얼은 부담없이 새로운 친구를 사귈 수 있는 교내 익명 랜덤 채팅 어플입니다.  
모르는 사람과 부담없이 친해질 수 있고, 여러가지 잡담, 고민 등을 이야기하며  
스트레스를 해소하는 것을 목적으로 만들어진 프로젝트입니다.
